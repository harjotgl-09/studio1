# **App Name**: Voicecribe

## Core Features:

- Audio Recording: Enable users to record audio directly within the app using the device's microphone.
- Transcription Processing: Send the recorded audio to the model (specified by github.com/jmaczan/asr-dysarthria) for transcription.
- Transcription Display: Display the transcribed text clearly on the app's home page.
- Text-to-Speech Playback: Implement a text-to-speech feature to allow users to play back the transcription.

## Style Guidelines:

- Primary color: Deep purple (#6750A4) for sophistication.
- Background color: Very light purple (#F3EFF7) creates a calming and focused environment.
- Accent color: Soft blue (#A0C5F8) to provide a gentle highlight to interactive elements and important text.
- Font: 'Inter', a sans-serif typeface, to promote readability.
- Use minimalist icons for recording, playback, and settings, focusing on clarity and ease of understanding.
- Design a clean and intuitive layout, prioritizing the transcription display on the home screen. The record and playback options should be easily accessible.
- Incorporate subtle animations to provide feedback during recording and playback actions.
